## Project Overview

This project analyzes retail sales data from the Superstore dataset to identify sales trends and business performance over time. The analysis focuses on monthly and yearly sales patterns using Python-based data analysis techniques.

## Objective

* Clean and preprocess sales data
* Analyze monthly sales trends
* Compare yearly sales performance
* Generate visual insights for business decision-making

## Dataset

Superstore Sales Dataset

Features used:

* Order Date
* Sales
* Category
* Region
* Profit

## Key Insights

### Monthly Trend

* Sales generally increase over time.
* Strong sales peaks are observed during the fourth quarter (October–December).
* Seasonal purchasing behavior is visible in the dataset.

### Yearly Trend

* 2015 showed a slight decline compared to 2014.
* Sales recovered significantly in 2016.
* 2017 recorded the highest sales performance.
* Overall sales demonstrate a positive growth trend.

## Conclusion

The analysis reveals a consistent long-term increase in sales with noticeable seasonal patterns. These insights can help businesses improve inventory planning, marketing campaigns, and revenue forecasting.
